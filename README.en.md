# Bitwarden Desktop (LoongArch64)

Build Bitwarden Desktop for LoongArch64 architecture.

This project depends on [bitwarden-loongarch64-binaries](https://github.com/miwu04/bitwarden-loongarch64-binaries) for pre-built native modules.

## Usage

Trigger the workflow manually via GitHub Actions with a version tag input.

## Packages

- `.deb` - Debian/Ubuntu
- `.rpm` - Fedora/RHEL
- `.pkg.tar.zst` - Arch Linux
